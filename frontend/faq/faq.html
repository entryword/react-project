PyLadies 是？
PyLadies 是一個國際性的 Python 交流組織，主要任務是希望透過分享、教育與 Workshop 等活動的方式幫助更多女生可以在 Python 社群中成為主動的參與者與領導者。而 PyLadies Taiwan 是由一群喜歡 Python 的志工組成的，我們希望建立一個專屬於女生的 Python 愛好者聚會，打造讓喜歡 Python 的女生、想學 Python 的女生、想瞭解 Python 的女生，彼此交流、交換心得，聊天認識朋友的友善環境。
所以 PyLadies 是一個授課/外包/家教的單位嗎？
並不是喔！PyLadies 是由一群願意奉獻自己時間精力的志工所組成，我們想分享各種的技術經驗給更多人而舉辦的活動，並不是以營利為目的導向的課程或接專案方式進行。
活動會收費嗎？
雖然 PyLadies 是非營利組織，但為了社群的營運，與活動的講師內容場地的不同，會酌收不等的費用，每次活動的詳細內容都可以參考粉絲專頁的宣傳和報名頁上的資訊。(ps. 無法開收據喔～)
怎麼參加 PyLadies 主辦的活動？
我們會定期舉辦分享聚會活動，不論是對完全沒碰過程式或是 Python 的初學者，或是想要更加進步的 Python 開發者，只要你是 Lady，都可以來參加喔！
每次活動內容、報名方式等詳細資訊，都會發布在官網的活動頁， Facebook 粉絲頁也會進行宣傳。如果需要第一手資訊，可以追蹤我們的紛絲專頁，而如果想要了解詳細資訊，請仔細瀏覽官網活動內容。
請詳閱PyLadies的報名管道教學與注意事項：Meetup、Accupass
怎麼參加 PyLadies 合辦、受邀的活動？
PyLadies 會跟很多不同的單位合辦或協辦活動，當主辦單位不是 PyLadies 時依主辦單位規定進行，報名相關資訊請參閱主辦單位的公告，或是詢問主辦方。
哪些活動是一系列的？是否可以中間才參加？
以「迴圈、字典」活動為例，可以看到有紅字「Python入門」的部分，點下去就可看到：此系列其他日期的活動以及已釋出的投影片。 系列活動有標數字編號的通常是連貫的活動內容，如果中間參加的話，建議先閱讀前幾次活動的投影片，參加活動才會比較有收穫喔～
投影片會公布嗎？有沒有影音檔呢？
目前PyLadies並沒有提供錄影或直播，而投影片只要徵得講師許可，都會公開在該次活動頁中。
好想參加上期活動，可以再開嗎？會在北部以外舉辦嗎？
因為PyLadies 是由一群願意奉獻自己時間精力的志工所組成，成員的生活範圍也以北部為主，在資源有限的情況下，會按照當期(一年為期)的規劃舉辦活動，可以從最新的主題、活動頁看是否有相關資訊，或是瀏覽前幾期的活動投影片。
為什麼 PyLadies 主辦的活動僅限女性參加？
因為在資訊科技業中，女性的比例都是很低的，不論在台灣或是歐美國家，讓許多想學程式、或是對於程式技術更精進的女性，在大多是男性的學習環境下，會顯得非常不自在，就如同男性踏入幾乎以女性為主的瑜伽、芭蕾課程的感覺因此我們希望打造一個能讓喜歡 Python 或是想學 Python 的女生，一個可以自在、安心學習交流的環境，促使更多女性投入資訊科技領域。
我是男性，有其他資源推薦的嗎？
PyLadies的活動投影片只要講師許可，都會公開；也有些 PyLadies 合辦、受邀的活動，男性也是可以參加的，請參閱主辦單位的公告。學習資源、各種社群可以參考此篇文章、或是我們定期發佈的電子報文章
開發上遇到問題，可以詢問PyLadies嗎？
PyLadies 的社團提供女性安心發言的空間，或是有不少網路社群可以發問。如果是看到PyLadies Medium的問題，請直接在該篇Medium文章下方留言，讓作者看到方便回覆喔～
Python有什麼魅力？
Python 是目前最熱門的程式語言之一，「優雅」、「明確」、「簡單」的設計哲學，很適合初學者學習，在各領域也能看到他的蹤跡，尤其近年資料相關領域正當時下話題，Python就是最常用的語言。
此外，Python 坐擁廣大的資源庫，讓我們不需要自己重新撰寫龐大的數學算式或是邏輯架構，即可輕鬆地進行資料分析、架設網站呈現各種視覺化報表、網路爬蟲、軟硬整合或是解決各種生活上瑣碎的重複工作等等的許多需求，都可以用 Python 這單一的語言完成，這也是 Python 如此受歡迎的原因。
